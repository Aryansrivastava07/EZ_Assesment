# V Films Website

A responsive single-page React application for V Films with integrated contact form.

## Features

- Fully responsive design (480p, 720p, 1080p, iPad, MacBook)
- Contact form with validation
- API integration with backend
- Modern UI matching Figma design

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will open at `http://localhost:3000`

## API Information

- **Endpoint**: https://vernanbackend.ezlab.in/api/contact-us/
- **Method**: POST
- **Required Fields**: name, email, phone, message

## Responsive Breakpoints

- Mobile: 480px
- Tablet: 720px
- Desktop: 1080px
- iPad: 2732x2048
- MacBook: 1440x823

## Build

```bash
npm run build
```
